# url---to---qr---converter🚀
"A Python-based utility to instantly convert any URL  into a high-quality QR Code image using the qrcode library."

## ✨ Features
- **Instant Generation**: Converts any URL into a scan-able QR Code.
- **Customizable**: Saves output as a clear `.png` image.
- **Lightweight**: Uses Python's reliable `qrcode` library.

## 🛠️ Tech Stack
- **Language**: Python 3.x
- **Library**: `qrcode` (with `Pillow` support)
